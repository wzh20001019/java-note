#### 1. application.yml 配置

```yml
# 服务器端口
server:
    port: 8000

spring:
    # 整合 mybatis
    datasource:
        driver-class-name: com.mysql.cj.jdbc.Driver
        url: jdbc:mysql://localhost:3306/db1
        username: root
        password: 12345

    # 修改banner (输出log)
    main:
        banner-mode: off

# 日志
logging:
    level:
        root: info
```
