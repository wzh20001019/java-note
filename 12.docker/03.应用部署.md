#### 1. 部署 mysql

```txt
// 1. 下载
docker pull mysql:版本

// 2. root下 创建mysql文件夹
mkdir mysql
cd mysql

// 3. 配置数据卷    $PWD(当前文件夹: root)
docker run -id \
-p 3306:3306 \                              容器3306 映射到宿主机 3306
--name=my_mysql \
-v $PWD/conf:/etc/mysql/conf.d \
-v $PWD/logs:/logs \
-v $PWD/data:/var/lib/mysql \
-e MYSQL_ROOT_PASSWORD=123456 \             配合 root 密码
mysql:5.6
```
