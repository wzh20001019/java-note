#### 1. 基本数据类型

```txt(整数)
byte         Byte              字节

short        Short

int          Integer           整数类型 (占4个字节)

long         Long              长整数类型 (占8个字节) (数字后加 L / l,  例: long num3 = 12859709462L;)
```

```txt(浮点数)
double       Double           小数类型 (占8个字节)

float        Float            单精度小数类型 (占4个字节) (数字后加 F / f,  例: float num4 = 10.5F;)
```

```txt(字符)
char         Character        字符类型(单引号)
```

```txt(布尔)
boolean      Boolean          布尔类型
```
